## 1. 회사 컨텍스트

- 2017년 창업, 9년 차 **스킨케어 D2C** 회사. 선케어 본격 진출 3년.
- 선케어 성과: "시작 3개월 만에 월 1억 → 1년 만에 120억" (자막 기준; 정확 수치는 재확인 필요)
- **사내 기술자(개발자)가 없음** — "완전히 시스템을 처음부터 만들고 설계할 수 있는" 상태
- 대표 직속 팀 구축 + 조직 개편 예정, 이번 컨테스트로 **6명 정도 채용 의사**
- 대표의 AI 사용: GPT(브레인스토밍) + Codex(코드 리뷰)
- 확장 비전: 스킨케어 → 헤어·향수 등 멀티브랜드. "온톨로지만 잘 구축하면 멀티브랜드 운영 가능"

## 2. 대표가 말하는 AX의 정의 (채점 철학의 힌트)

자동화 ≠ AX. 대표의 4단계 정의:
1. 과거 사람이 내리던 **판단을 시스템이 내리는가**
2. 그 판단으로 **매출이 났음을 증명**할 수 있는가
3. 시스템이 **더 똑똑해지고 있는가**
4. 사람 없는 새 시장에도 시스템으로 진출해 성과가 나는가

→ 결과물이 "자동화 도구"에 머물면 감점 요인. **판단 → 매출 증명 → 학습 루프**의 구조를 갖춰야 대표의 AX 정의에 부합.

사람 대비 AI의 장점으로 꼽은 것: ① 히스토리 전부 누적(퇴사·기억 한계 없음) ② 편견 없이 다변인 동시 분석·가설 검증.

## 3. 본선 문제 — 대표가 직접 밝힌 구조

공식 문제: **"매출이 될 수 있는 인플루언서 시딩 시스템을 설계해라"**

"화장품 회사 마케팅 문제가 아니라" 구조적으로 3가지로 분해된다고 직접 언급:

| # | 과제 | 대표의 설명 |
|---|---|---|
| 1 | **온톨로지 구축** | 인스타·틱톡·페북·유튜브 SNS 데이터의 "세계관 정의". 콘텐츠→구매 사이에 수많은 어트리뷰션 관여가 일어나므로 이 세계를 먼저 정의 |
| 2 | **인과관계 추정** | "소비자가 A·B·C 광고를 보고 구매했을 때 뭐가 진짜 기여했는지는 **어떤 회사도 해결 못 한 기술적 난제**" — 인과를 추정하고 그 근거를 증명 |
| 3 | **반복 가능한 루프** | 증명된 인과를 계속 돌릴 수 있는 루프로 설계 |

대표의 요약: "온톨로지, 인과관계 추정, 루프 만들기 = **매트릭스(현실 시뮬레이션)를 설계하는 일**". 100% 해결 안 돼도 일부만 접목해 마케팅 과제를 풀면 엄청난 부가가치.

## 4. 평가 기준 — 명시적으로 2개

1. **문제 정의 능력** (가장 중요) — "모든 데이터를 다 가져올 수 없으니, **비즈니스 임팩트가 가장 높은 것을 선택하는 기준** 자체가 문제 정의"
2. **창발성** — AI로 솔루션을 설계하는 과정에서 인간이 기여하는 창의적 접근

→ 예선 강점과 정확히 일치: 예선에서 통한 "공개 근거로 문제를 좁히고 스스로 반증하는" 방식이 곧 문제 정의 능력의 증거. 브리프의 ②문제 정의 항목에 화력을 집중해야 함.

## 5. 본선 당일 요구사항 예측

영상 근거로 예측(당일 상세 공개 전 가설):

- **산출물 형태**: 3시간 안에 "시딩 시스템 설계 + 동작하는 데모". 완제품보다 ①온톨로지 스키마 ②인과 추정 로직 ③루프 구조가 보이는 수직 슬라이스가 유리
- **데이터**: 현장 제공 가능성(메디테라피 시딩 이력/판매 데이터) 또는 공개 SNS 데이터 수집. 어느 쪽이든 "어떤 데이터를 왜 선택했나"가 채점 포인트
- **매출 연결**: "매출이 될 수 있는"이 문제 제목에 들어감 — 시딩 → 콘텐츠 → 유입 → 구매 전환의 연결고리를 수치로 시뮬레이션이라도 해야 함
- **함정**: 인플루언서 리스트업 자동화 툴(발굴·컨택 자동화)에만 머물면 대표가 말한 "자동화 ≠ AX"에 걸림

## 6. 전략 시사점 (다음 문서에서 구체화)

1. 3시간 배분: 문제 정의·온톨로지 설계에 앞부분을 과감히 투자 — 채점 1순위가 문제 정의
2. 예선 방식 재사용: 결정론적 코어(스코어링·규칙) + AI는 주변(수집·해석) — "증명 가능"이라는 대표 요구와 맞음
3. 12:00 제출(선착순 심사) 목표 → 사전에 스캐폴드(데이터 모델·대시보드 뼈대·시뮬레이션 루프)를 준비하고 당일은 요구사항에 맞춰 조립
4. 1대1 시연에서 대표의 언어로 말하기: "판단을 시스템이 내리고, 매출 증명이 되고, 루프가 돌아 똑똑해진다"
