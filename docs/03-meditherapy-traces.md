# 메디테라피 시딩 흔적 수집 — 리허설 기록 (2026-07-16)

목적: 본선 당일 재실행할 "브랜드의 기존 시딩 이력 수집" 과정을 미리 한 바퀴 돌려봄.
당일에는 브랜드/키워드만 바꿔 같은 과정을 반복한다.

## 재실행 절차 (당일용)

1. `python tools/collect_youtube.py "<브랜드>" "<브랜드> <주력제품>" "<브랜드> 리뷰"` → `data/youtube-mentions.csv`
2. 웹 검색으로 네이버 블로그·체험단 흔적 확인
3. 자사몰 리뷰 구조 확인 (리뷰 인센티브 프로그램 유무)
4. 인스타그램은 로그인 장벽 → 당일 수동 확인 or 제공 데이터 사용 (아래 한계 참고)

## 수집 결과 1 — 유튜브 (60건 검색, 브랜드 직접 언급 16건)

원본: [data/youtube-mentions.csv](../data/youtube-mentions.csv)

관찰된 시딩/콘텐츠 유형 (온톨로지의 "콘텐츠 유형" 축 후보):
| 유형 | 예시 | 시사점 |
|---|---|---|
| 커머스쇼 협찬 | 선데이미디어 "트렌드쇼퍼" 2건 (2.5만~2.9만 views) — 포쎄라 비피다 크림, 필리지오 볼륨 펌핑 | 브랜드가 이미 유료 협찬 집행 중 |
| 내돈내산 리뷰 | 오지월드 링클핏 마스크 후기 (5.9만 views), 보아따 레티날 세럼 | 오가닉 리뷰 존재 = 제품력 신호 |
| SNS광고템 검증 리뷰 | 아임깡지 "릴렉스패치 광고처럼 붓기 빠지냐" | SNS 광고를 많이 태운 흔적 (소비자가 검증 콘텐츠를 찾음) |
| 셀럽 콘텐츠 | 글램미 솔라 편 (모톡스/톤업크림 노출) | 셀럽 PPL 이력 |

제품 라인 파악(영상 제목 기준): 링클핏 마스크, 포쎄라 비피다 크림, 레티날 세럼, 모톡스 텐션업(예선 처분 대상 제품), 속살괄사·속살패드, 릴렉스패치, 필리지오(디바이스). → 스킨케어 + 뷰티 디바이스 혼합 D2C.

특이: 조코딩 AX 인재전쟁 1화 영상(7.4천 views)도 검색에 잡힘 — 본선 문제 맥락 그 자체.

## 수집 결과 2 — 자사몰·기타

- 자사몰(meditherapy.co.kr)에 **"베스트 리뷰어" 적립금 프로그램** 운영 중 — 리뷰 UGC 인센티브가 이미 제도화됨. 시딩 엔진의 "검증 단계 콘텐츠 축적"과 연결 가능
- 네이버 블로그: 미국 기반 검색으로는 개별 협찬 후기 직접 확인 못 함 → **당일 네이버에서 직접 검색 필요** ("메디테라피 협찬", "메디테라피 체험단")
- 블라인드에 회사 리뷰 존재 (조직 참고용)

## 정직한 한계 (당일 심사 답변용)

1. **인스타그램 해시태그 데이터는 로그인 장벽으로 자동 수집 불가** — 공식 API(그래프 API)는 비즈니스 계정 권한 필요. 당일: 수동 샘플링(브라우저에서 #메디테라피 상위 게시물 기록) 또는 주최 측 제공 데이터 사용
2. 유튜브 검색은 상위 노출분만 — 전수가 아니라 표본
3. views는 수집 시점 스냅샷

## 온톨로지 시사점 (다음 문서로)

수집해보니 실제 온톨로지 축이 보임:
- **인플루언서**: 채널, 구독자, 콘텐츠 유형(커머스쇼/리뷰어/일상)
- **콘텐츠**: 협찬 여부(내돈내산 vs 광고), 형식(쇼츠/롱폼), 제품 노출 방식
- **제품**: 라인별(마스크/세럼/디바이스), 가격대
- **전환 신호**: 코드/링크(당일 설계), 리뷰 축적(자사몰 베스트리뷰어)
